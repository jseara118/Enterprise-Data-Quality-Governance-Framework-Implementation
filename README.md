# Enterprise-Data-Quality-Governance-Framework-Implementation
This project demonstrates the design and implementation of an enterprise-level Data Quality and Data Governance framework applied to a simulated production environment.  The solution addresses unreliable data generated by an application and consumed by BI and downstream systems.
